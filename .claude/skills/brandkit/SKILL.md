This skill turns a short briefing into a presentable brand board with logo ideas, a color system, typography, and mockups.

## When this skill activates
- A new product, sub-brand, or company needs a first visual concept ("Create a brand board for a B2B IT consultancy").
- Logo and color proposals are needed before a designer takes over.
- An existing brand should be documented as one board with color and font specs.

## Workflow
1. Clarify the briefing: industry, target audience, competitors, and 3 brand attributes (e.g. precise, approachable, technical). Fill gaps with plausible assumptions and mark them.
2. Write the positioning in one sentence: "For [audience], [brand] is the [category] that [differentiator]." Every design decision must serve this sentence.
3. Sketch 2 to 3 design directions (e.g. geometric minimal, editorial warm, technical dark) and recommend one with reasoning.
4. Build the color system: 1 primary, 1-2 secondary colors, 2 neutrals, 1 accent. All values as HEX with a stated job. Verify contrast mathematically (body text at least 4.5:1).
5. Set typography: 1 headline font plus 1 text font from freely licensed sources (e.g. Google Fonts), with a fallback stack and a size scale (e.g. factor 1.25).
6. Develop 3 to 5 logo concepts: wordmark first, a symbol only if the idea carries weight. Test each for single-color use and legibility at 24px.
7. Show mockups: website hero, business card, social profile image, each using the defined colors and fonts.
8. Assemble everything on one single-page brand board (HTML or SVG) and check it against the quality rules.

## Output format
- One single-page brand board (HTML or SVG) with logo variants, color palette, font pairing, and mockups.
- A compact text spec: positioning sentence, color table (HEX, purpose), fonts with license note, logo rationales, 3 don'ts.

## Quality rules
- Every color has a HEX value and a named job, no decorative extras.
- Body text reaches at least 4.5:1 contrast, large headlines 3:1.
- No more than 2 type families and 6 colors including neutrals.
- Only suggest fonts with a clear free license or an existing client license.
- No industry clichés (lightbulb, gear, handshake) without a deliberate twist.
- The logo works in one color, inverted, and as a 24px favicon.
- Every design decision gets a one-sentence rationale, not just an assertion.
